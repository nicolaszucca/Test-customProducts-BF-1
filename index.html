Documentacion

  1. Se debe seleccionar perimero un elemento (texto, fecha, imagen) para aplicar una funcionalidad.
  2. Los botones debajo de la funcionalidad "espejo" son para el sistema de capas.
    El boton superior lleva al elemento que hayas seleccionado una capa hacia "arriba".
    El boton inferior lleva al elemento que hayas seleccionado una capa hacia "abajo".
  3. Los "circulos" estan especificamente añadidos para testear el sistema de capas con elementos distintos.
    No fueron tenidos en cuenta como funcionalidad específica.
  4. Para crearle un contorno al elemento seleccionado, el tamaño del mismo va a estar establecido por el input y su valor correspondiente, el cual se encuentra a la derecha.

    En el canvas, se debe seleccionar el elemento vacío (sin ningun contorno), establecer un tamaño numerico y luego aplicarlo con la funcionalidad "crear contorno"
